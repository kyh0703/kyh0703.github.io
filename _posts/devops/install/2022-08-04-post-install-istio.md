#### IstioCtl

클러스터에 istio를 설치할 때 사용

[설치링크](https://istio.io/latest/docs/setup/getting-started/#download)

> 알파나 베타는 설치하지 않는다.

* istio 설치

```bash
istioctl install
```

* 실험 명령어 보기

```bash
istioctl x
```
